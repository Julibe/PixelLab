### THE GENESIS BIT (Theory)
The source code was forged in the fires of a specific design challenge: **CROSSOVER CHAOS**. The mission? Merge the lane-hopping logic of **[Frogger](https://en.wikipedia.org/wiki/Frogger)** with the punish-physics of **[Flappy Bird](https://en.wikipedia.org/wiki/Flappy_Bird)**.

Initially, it was just a frog trying to fly. **BORING!** I flipped the axis to VERTICAL and suddenly, we had lift-off! But a rocket needs enemies, right? The visuals shifted to space, channeling the spirit of **[Galaga](https://en.wikipedia.org/wiki/Galaga)**. That demanded firepower!

Finally, to inject that pure arcade adrenaline, I sprinkled in some **[Contra](https://en.wikipedia.org/wiki/Contra_(video_game))** DNA. Now you aren't just dodging; you're collecting Spread Guns, Nukes, and Shields! It evolved from a physics test into a **HYBRID ARCADE MONSTER**.

### SYSTEM GLITCHES (Challenges)
*   **PHYSICS ANOMALIES:** Balancing the "Flap" gravity with "Frogger" lanes was trickier than beating **[Mike Tyson](https://en.wikipedia.org/wiki/Punch-Out!!_(NES))** on the NES! Players would fall into the void or warp off-screen. I had to tune the gravity constants to perfection.
*   **PROCEDURAL CHAOS:** Random generation can be unfair! I had to program the "Director AI" to ensure it didn't spawn an impossible wall of death at level 1.
*   **SYNTHESIS OVERLOAD:** I refused to use pre-recorded MP3s! Every *pew-pew* and *boom* is generated by the **Web Audio API** in real-time. Creating 8-bit explosions using only math waves was and always is a serious headache!
*   **INPUT LAG:** Merging digital keyboard inputs with analog gamepad sticks required a custom polling system to ensure ZERO LATENCY.
*   **STAR PARALLAX:** Those stars refuse to move under my rule! I made them pay!

### TURBO BOOSTS (Solutions)
*   **SMART CAMERA:** Implemented a scrolling camera that locks the "death zone" at the bottom. If you drop below the screen, you're space dust!
*   **LEVEL SCALING:** The `createLane` algorithm reads your `altitude_count`. It starts with rocks, then evolves to UFOs and Tanks as you climb higher. It learns your skill level!
*   **CUSTOM SOUND CHIP:** Built a virtual sound synthesizer (`sfx_engine`) that modulates oscillators to recreate that crunchy, warm NES soundscape.
*   **CONTROLLER FUSION:** A unified input manager (`keys_state`) polls every frame, letting you switch from Keyboard to Controller instantly without pausing. **SEAMLESS ACTION!**

## UNDER THE HOOD (Technical Details)
Listen up, hacker! This isn't built with bloatware engines. This is **BARE METAL JAVASCRIPT**. The entire game runs on a custom **Entity Component System (ECS)** loop rendering at a blistering 60 FPS via `requestAnimationFrame`.

Every pixel you see is drawn via code (`ctx.fillRect`). No sprites. No loading bars. Just pure, unadulterated rendering power. The "Lanes" system generates horizontally but scrolls vertically, recycling memory like a responsible eco-warrior to keep your frame rate high!

### POWER-UPS (Technologies)
This game runs on **PURE WEB POWER**:
*   **[HTML5 Canvas](https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API)**: For pixel-perfect, high-speed graphics.
*   **[Web Audio API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API)**: Real-time synthesis for that authentic chiptune crunch.

*   **[SVG Filters](https://developer.mozilla.org/en-US/docs/Web/SVG/Element/filter)**: Creates the retro CRT glow and scanlines.
*   **[LocalStorage](https://developer.mozilla.org/en-US/docs/Web/API/Window/localStorage)**: Saves your legendary scores forever.
